# Changelog

All notable changes to Predictify Numbers will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added
- Initial release of Predictify Numbers
- Number guessing game with Bulls and Cows mechanics
- 4 difficulty levels: 2, 3, 4, and 5 digit modes
- Real-time game timer and move counter
- Custom number input keypad
- Guess history with Bulls/Cows feedback display
- Win celebration with confetti animation
- Persistent leaderboard with top 10 records per difficulty
- Medal display for top 3 records
- Haptic feedback on button interactions
- Shake animation for invalid input
- Dark theme UI
- Portrait-only orientation
- Full offline support
